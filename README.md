

# **Financials Report – Power BI Dashboard**

This repository contains a practice **Power BI Financials Report** built to explore revenue trends, cost patterns, and overall financial performance. The dashboard provides an easy-to-understand view of financial KPIs through interactive visuals.

## **Features**

* Revenue, cost, and profit analysis
* Monthly and category-level insights
* Interactive filters and drill-through options


## **File Included**

* `FINANCIALS REPORT PRACTICE.pbix` – Power BI practice dashboard

## **How to Use**

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. Explore the visuals using slicers and filters

## **Author**

Created as part of ongoing Power BI practice and financial analysis learning.
